HTML : hyper text markup language
태그는 MDN에서 공부

-기본구조(vscode 단축키 =  !+tab)
    <!DOCTYPE html>
    <html lang="en">
        <head>
            <meta charset="UTF-8" />
            <meta name="viewport" content="width=device-width, initial-scale=1.0" />
            <title>Document</title>
        </head>
        <body>

        </body>
    </html>

html 태그 안에 head,body가 하나씩 필요함.

<ul> - 순서없는 리스트
<ol> - 순서있는 리스트
    <li> - 리스트 내용

<a href="주소"> </a> - 링크

<img src="경로">
    1) 이미지 경로가 문서파일과 동일한 위치 -> 파일명
    2) 하위 폴더에 위치 -> 하위폴더/파일명
    3) 웹상의 이미지 -> 이미지 주소복사

    + alt 속성을 이용해 이미지 표시 전에 뜨는 글자를 나타낼 수 있음.

주석: <!-- 내용 -->
