css - cascading style sheets

link 태그를 이용하여 css파일을 html파일에 연결시킨다. (헤드에 포함되어야함)
    ex) <link rel = "stylesheet" href= "style.css" />


색상
    - RGB 
    - 16진법(Hex)   ex) #4477ff 순서는 rgb와 동일함.
    (구글 color picker로 편하게 찾을 수 있음)
    (coolors 사이트에서 감각적인 색들 찾을 수 있음)


텍스트 속성
    - text-align : 텍스트 정렬
    - font-weight : 텍스트 굵기 설정
    - font-size : 텍스트 크기 설정
        절대단위 px를 보통씀
        상대단위로는 vh.
    - text-decoration : 텍스트 꾸미기

    -font-family : 폰트 설정
        폰트를 여러개 지정하는 이유는 지정 글꼴이 없을때를 대비한것임.
