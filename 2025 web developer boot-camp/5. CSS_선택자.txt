*** Selector

    * : 전체 선택자
    # : ID 선택자
    . : Class 선택자
    
    하위 선택자: 상위요소 하위요소
    자식 선택자: 부모요소 > 자식요소 (바로 밑 요소를 의미함)
    인접 형제선택자: 요소1 + 요소2
        ex) h1+P 라고 하면 h1과 같은 레벨중 맨처음 p에 스타일 적용됨.

    속성 선택자
        
        [속성="값"]
        [속성~="값"] : 값이 포함되면 적용 (독자적인 단어)
        [속성*="값"] : 값의 일부가 일치하는 속성에 적용
        [속성^="값"] : 특정값으로 시작하는 속성에 적용
        [속성$="값"] : 특정값으로 끝나는 속성에 적용
        (더 많지만 일반적으로 거의 안씀)

    **Pseudo 선택자(세부적으로 요소 선택)

        [사용자 동작]
            :hover - 마우스 커서를 올려놓았을때
            :active - 웹요소를 활성화 했을때 (누르고 있을때)
            :visited - 방문한 링크

        :first-child
        :nth-child()
        :nth-of-type()

    pseudo elements
        ::first-letter - 첫글자만 선택
        ::first-line - 첫줄만 선택
        ::placeholder - placeholder 선택
        ::selection - 드래그로 선택되었을때

적용범위 중요도
    1. 인라인스타일 ex) style 속성
    2. ID
    3. Class
    4. 요소태그

    !important 를 쓰면 최우선으로 적용된다.
        ex) color : green !important;

상속
    - 상위요소의 속성을 상속하려면 inherit을 쓰면됨
        ex) color : inherit;
        
    - 속성에따라 상속할 수 있는지 유뮤가 다름.
