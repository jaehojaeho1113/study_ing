  
    컨텐츠 영역
        width : 가로
        height : 세로

    border
        border-width : 테두리 굵기 설정
        border-color : 테두리 색 설정
        border-style : 테두리 스타일 설정
        border-radius : 모서리 둥글게 설정
        border (굵기, 색, 스타일)로 한번에 지정 가능.

    * box-sizing - 박스 너비 기준 정하기
        content-box
        border-box : border 굵기를 content 크기에 포함시킴.

    padding : 박스와 콘텐츠 사이의 공간
    margin : 여러 박스 모델 사이의 공간

영역
    margin - border - padding - contents

    -블록레벨 : 옆에 다른요소 x (대부분)
    -인라인레벨 : 옆에 다른요소 가능 (span, a, image 등)
        * 높이랑 너비가 없음. 수직 margin,padding 적용 불가함.
    
    display: inline-block 으로 블록레벨 요소를 인라인레벨처럼 쓸 수 있음
    display: none으로 안보이게 설정 가능함.

단위
    상대단위
        % : 부모요소를 기준으로 %적용함.
        em : 부모요소의 글꼴크기를 1em으로 적용함.
        rem : em의 단점을 보완. 루트 글꼴 크기를 1rem으로 적용함.
